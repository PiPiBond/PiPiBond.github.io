                # Windows 安装

                下载时选择不登录直接下载，下面将展示如何安装绿色版 (MySQL Community Server) 和安装版 (MySQL Installer for Windows)

                # 绿色版 (MySQL Community Server)

                # 安装 MySQL

                1. 绿色版下载的是一个压缩包，没有 图形界面

                解压后，文件结构如下

                安装完成后根目录应该是下图所示

                1. 需要自己在 MySQL 的根目录 创建 一个 my.ini 的文件，基本内容如下，此配置基本满足学习需要。
                [client]

port=3306

default-character-set=utf8

[mysql]

default-character-set=utf8mb4

[mysqld]
# 手动设置默认端口
port=3306
# MySQL的根目录，该路径填写 MySQL 的绝对路径，确保程序的稳定
basedir=/Environment/mysql-8.x.xx
# 数据目录，建议直接放在根目录下，方便维护
datadir=/Environment/mysql-8.x.xx/data

character-set-server=utf8

default-storage-engine=INNODB
                1. 以管理员身份打开终端，进入 bin 目录，执行命令。
                mysqld --initialize --console
# 初始化MySQL数据库，会生成几行English，root@localhost: 后面是密码
# A temporary password is generated for root@localhost: vp9i%z!0rl#E
                1. 安装 MySQL 服务
                mysqld --install #MySQL 默认服务名，如果电脑同时安装不同版本的服务，可以在此自定义服务名
                1. 启动服务
                net start mysql
                1. 进入 MySQL 命令行
                # 明文输入密码
mysql -u root -p<密码> # p与密码之间不能有空格

# 回车后输入密码
mysql -u root -p
#Enter password: ************
                1. 修改密码
                ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '你的密码';

                # 删除 MySQL

                net stop mysql
# MySQL 服务正在停止.
# MySQL 服务已成功停止。
sc delete mysql
# [SC] DeleteService 成功

                进入 MySQL 的根目录上一层 Environment，选中 mysql-8.x.xx，按住键盘 Shift + Del，回车。

                # 安装 MySQL

                1. 双击下载好的 .msi 文件，以管理员运行，选择 Custom 安装

                1. 将 MySQL 服务选择到下载列表，单击右边的 MySQL Server 8.0.32 - X64 ，出现蓝色字体 Advanced Options ，单击。

                1. 将数据目录修改为 MySQL 的根目录下，点击 OK。

                1. 下一步直到如图所示

                选择修改端口 Port，或者保持默认。

                1. 点击下一步直到提示输入密码界面，输入两次密码。

                1. 修改 MySQL 服务名或保持默认

                1. 连续点击下一步，安装完成。

                # 删除 MySQL

                1. 双击 .msi 安装文件

                1. 勾选当前卸载项

                1. 勾选删除数据文件
